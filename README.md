# ポケコン PC-G850用ゲーム 『平城京エイリアン　heijo』

## 動作機種
SHARP PC-G850系

## 開発言語
C（Z88DK）＋機械語

## 同梱ファイル

| ファイル名  | 内容                                |
|-------------|-------------------------------------|
| `README.md` | マニュアルファイル                  |
| `data.hex`  | データファイル（Intel HEX形式）     |
| `heijo.hex` | プログラムファイル（Intel HEX形式） |

## セットアップ

本体へ、以下の順でセットアップします。

### 実機で、PC-Gリンク等の転送ソフトをお使いの場合

* BASICモードから`MON`で機械語モニタを起動し、`USER 40DF`と入力してEnterキーを押し、機械語のユーザーエリアを確保します。
* `R`と入力してEnterキーを押すと、プログラムファイル（`heijo.hex`）が転送されます。
* `R`と入力してEnterキーを押すと、データファイル（`data.hex`）が転送されます。

### SHARP PC-G850/G815/E200エミュレータ g800をお使いの場合

#### 実機ROMファイルがある場合

* g800を起動します。
* BASICモードから`MON`で機械語モニタを起動し、`USER 40DF`と入力してEnterキーを押し、機械語のユーザーエリアを確保します。
* エクスプローラーからg800のウインドウへ、プログラムファイル（`heijo.hex`）をドラッグ&ドロップすると、g800へ転送するファイルがセットされます。
* `R`と入力してEnterキーを押すと、ドロップしたファイルが転送されます。
* エクスプローラーからg800のウインドウへ、データファイル（`data.hex`）をドラッグ&ドロップすると、g800へ転送するファイルがセットされます。
* `R`と入力してEnterキーを押すと、ドロップしたファイルが転送されます。

#### 実機ROMファイルが無い場合

* g800を起動します。
* BASICモードから`MON`で機械語モニタを起動し、`USER 40DF`と入力してEnterキーを押し、機械語のユーザーエリアを確保します。
* エクスプローラーからg800のウインドウへ、プログラムファイル（`heijo.hex`）をドラッグ&ドロップすると、ファイルが転送されます。
* エクスプローラーからg800のウインドウへ、データファイル（`data.hex`）をドラッグ&ドロップすると、ファイルが転送されます。

## 起動方法

BASICモードから`MON`で機械語モニタを起動し、`G100`で起動します。<br>
または、BASICモードから`CALL &H100`でも起動できます。<br>

## ゲームのジャンル

戦略系穴掘りアクション<br>

## ストーリー

なんと見事な平城京（710年）。<br>
天平の都・平城京に、どう考えても時代を間違えたエイリアンが出現した。<br>

都の警備を担う衛門府が出動するが、ここで深刻な事実が判明。<br>
プレイヤーは弱い。とにかく弱い。<br>
剣も槍も重すぎて、持った瞬間に腰を痛める。<br>

残された装備は、なぜか正式支給された一本のスコップ。<br>
衛門府は結論を出す。<br>
戦は腕力ではない。穴である。<br>

プレイヤーは今日も都を守るため、掘って、誘って、落として、埋める。<br>
平城京の平和は、あなたの筋力ではなく、穴の完成度にかかっている。<br>

## ゲームのルール

プレイヤーは、平城京を守る衛門府です。<br>
`上下左右`で移動します。<br>
斜め移動はできません。まっすぐ生きましょう。<br>

`スペースキー`、または `Zキー`で地面に穴を掘ります。<br>
`Xキー`、または `Cキー`で穴を埋めます。<br>
穴を掘るのも埋めるのも、あなたの仕事です。<br>
ステージには壁があります。<br>
壁は非常にかたく、越えることはできません。<br>
都の区画整理は完璧です。<br>

エイリアンはとても強く、直接触れると捕まります。<br>
捕まると喰べられてしまい、昇天します（ミス扱い）。<br>
人生、逃げるが勝ちです。<br>

３回捕まると、ゲームオーバーとなります。<br>
全２０ステージをクリアすると、エンディングです。<br>
都の平和と、あなたの努力が報われます。<br>

## 操作方法

### 《タイトル画面》

| キー | 役割 |
|------|------|
| `Sキー` | 音のOFF/ONの切り替え<br> ※本体に圧電ブザーが取り付けられていない場合、ONにしても音は鳴りません。 |
| `SPACEキー` | ゲームスタート |
| `Cキー` | ゲームオーバーとなった最後の面から、続行します。 |

### 《ゲーム》

| キー                       | 役割                           |
|----------------------------|--------------------------------|
| `８`・`２`・`４`・`６キー` | 衛門府を上下左右に操作します。 |
| `SPACEキー`または `Zキー`  | 地面に穴を掘ります。           |
| `Xキー`または `Cキー`      | 掘った穴を埋めます。           |

## キャラクター紹介

### 《プレイヤー（衛門府）》

あなたです。<br>
平城京を守るために配属された、衛門府の一員です。<br>

とても弱いので、剣も槍も使えません。<br>
走る、掘る、埋める、それだけが取り柄です。<br>
足は速くありませんが、仕事は地道です。<br>

エイリアンに触れると捕まってしまいます。<br>
捕まると喰べられて昇天します。<br>
勇気より距離感が大切です。<br>

### 《エイリアン》

平城京の地下から出てきた、よく分からない存在です。<br>
とにかく強く、直接倒すことはできません。<br>

追いかけるのは得意ですが、足元はあまり見ていません。<br>
穴に落ちると、動かなくなります。<br>

数が増えると、一気に危険になります。<br>
近づかせないことが、最大の対策です。<br>

## 終了方法

本体の`BASICキー`・`TEXTキー`・`OFFキー`・`Qキー`のいずれかを押してください。<br>

## うまく動作しない場合

一度、本体の電源を切り、再度起動してください。<br>

## その他

* 音を出したい場合、本体へ圧電ブザーを取り付ける必要があります。<br>
圧電ブザーの取り付け方法につきましては、[作者のWebSite](http://yswgt879.blog.fc2.com/blog-entry-360.html)か、各種サイトにてご確認ください。<br>
* SHARP PC-G850/G815/E200エミュレータ g800から動作させる場合、実機ROMが無いと、入力が若干遅かったり、音割れが発生する等の問題があります。<br>
音割れが気になるようであれば、サウンドはOFFにしてください。<br>
